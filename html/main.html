TabSentinel – Lightweight, Private, Open-Source Tab Suspension for Chrome

TabSentinel (Tab Sentinel) automatically suspends inactive tabs after a specified time, helping to reduce memory usage and improve browser responsiveness during long sessions.

Key Features: - Automatic tab suspension with configurable delay. - Exclusion support for selected websites. - Runs fully locally – no remote servers, no data transmission. - Open-source: github.com/dmk500/TabSentinel - No tracking, no ads, no analytics. - Minimal UI, focused on performance and reliability.

No account required. No background communication. All logic runs in the browser and can be verified or modified via source code.

    About Tab Sentinel

    Tab Sentinel is a lightweight Chrome extension that helps you improve browser performance by automatically suspending inactive tabs.

    This extension was developed by LLM LOUNGE and is 100% free and open-source. It contains no hidden functions, no analytics, and does not collect or share any user data.
