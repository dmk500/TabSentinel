Privacy Policy

Last Updated: February 2025

1. Introduction

Welcome to TabSentinel! This Chrome extension helps users improve browser performance by suspending inactive tabs. We value your privacy and ensure transparency about our extension's operation.

2. Data Collection and Usage

TabSentinel does NOT collect, store, or share any personal data.

The extension only stores user-defined preferences within the browser using Chrome Storage API, including:

  • Suspend time settings
  • Excluded websites

These settings are stored locally and are not shared externally.

3. Permissions and Justifications

TabSentinel requires the following permissions for its core functionality:

  • tabs: To detect inactive tabs and suspend them accordingly.
  • storage: To save user preferences.
  • scripting: To modify inactive tabs when suspended.
  • alarms: To periodically check for inactive tabs.
  • activeTab: To allow users to exclude a tab from suspension.
  • host_permissions: To apply suspension logic to all websites.

TabSentinel does NOT monitor browsing history or track user activity.

4. Third-Party Services

This extension does NOT use third-party services for analytics, advertisements, or data collection.

5. Security Measures

We prioritize security by:

  • Keeping all scripts bundled within the extension (no remote execution).
  • Using only necessary Chrome permissions.
  • Not requiring login or personal data input.

6. Updates & Changes

We may update this Privacy Policy to reflect changes in functionality or compliance. Any updates will be reflected on this page with a revised date.
